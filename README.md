# Film React

## Requirements

- Node.js (We recommend latest LTS)
- [yarn](https://yarnpkg.com)

## Installation

1. Install dependencies
    ```
    yarn
    ```

## Development

### Browsers

```
yarn start
```

> IMPORTANT: Make sure that `NODE_ENV` is set to `development`

## Lint

```
yarn lint
```

### Lint CSS only

```
yarn lint:css
```

### Lint JS only

```
yarn lint:js
```

## Format code

```
yarn format
```

## Test

```
yarn test
```

With watch mode:

```
yarn test -w
```


### Type checks

```
yarn flow
```

## Storybook

### Initialize server

```
yarn storybook
```

### Build

```
yarn storybook:build
```

## ESDoc

### Generate document

```
yarn docs:generate
```

### Initialize static server for ./docs

```
yarn docs:serve
```

### Generate docs and initialize server

```
yarn docs
```
